# DATA-INTEGRATION





Project Description

This project demonstrates data integration from multiple sources by combining Excel-based transactional sales data with SQL Server master data (Products & Regions) to create a unified Power BI report.

The dashboard enables analysis of sales trends, product performance, and regional performance, simulating a real-world enterprise BI scenario where data is spread across different systems.
